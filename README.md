📊 HR Analytics Dashboard (Power BI)
📌 Project Overview

The HR Analytics Dashboard is an interactive Power BI project designed to analyze employee data and identify workforce trends. It provides insights into attrition rate, employee demographics, job satisfaction, and department performance to support data-driven HR decisions.

🎯 Objectives

Analyze employee attrition and retention patterns

Understand workforce distribution by age, gender, and education

Evaluate job satisfaction across different roles

Help HR teams make strategic decisions using data visualization

📈 Key Features

Overall employee and attrition overview

Department-wise attrition analysis

Age group and gender distribution

Job satisfaction rating by job role

Education field-wise employee insights

Interactive filters and visual reports

🛠️ Tools & Technologies

Power BI Desktop

Data Visualization

Data Cleaning & Modeling

📂 Dataset

The dataset contains employee-related information such as:

Employee demographics

Department & job roles

Education background

Attrition details

Job satisfaction ratings

(Dataset used for learning and analytical purposes.)

📊 Insights Generated

Identified departments with higher attrition rates

Analyzed age groups most affected by attrition

Compared job satisfaction across roles

Highlighted workforce demographic trends
